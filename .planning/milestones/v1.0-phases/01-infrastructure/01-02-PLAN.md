<task type="auto">
  <name>Task 2: Deploy infrastructure to Kubernetes cluster</name>
  <files></files>
  <action>
Deploy everything to the cluster in dependency order. All commands run via kubectl/helm on the local machine targeting the Talos cluster.

**Pre-flight checks:**
1. Verify kubectl connectivity: `kubectl cluster-info`
2. Verify Helm is installed: `helm version` (if missing, install: `brew install helm`)
3. Verify NFS export is accessible: `showmount -e 192.168.0.102` (should show `/volume1/k8s`)

**Step 1: Create namespace**
```
kubectl apply -f k8s/namespace.yaml
```
Verify: `kubectl get namespace sketchpad`

**Step 2: Install NFS provisioner via Helm**
```
helm repo add nfs-subdir-external-provisioner https://kubernetes-sigs.github.io/nfs-subdir-external-provisioner/
helm repo update
helm install nfs-subdir-external-provisioner nfs-subdir-external-provisioner/nfs-subdir-external-provisioner \
  --namespace sketchpad \
  --set nfs.server=192.168.0.102 \
  --set nfs.path=/volume1/k8s \
  --set storageClass.name=nfs-client \
  --set storageClass.defaultClass=false \
  --set storageClass.reclaimPolicy=Retain \
  --set storageClass.archiveOnDelete=false \
  --set storageClass.accessModes=ReadWriteOnce
```
Verify: `kubectl get storageclass nfs-client` shows the StorageClass
Verify: `kubectl get pods -n sketchpad -l app=nfs-subdir-external-provisioner` shows Running

**Step 3: Create PVCs**
```
kubectl apply -f k8s/pvc.yaml
```
Verify: `kubectl get pvc -n sketchpad` -- both should become Bound (may take a few seconds)

**Step 4: Create Kubernetes Secrets**
Use the credentials provided by the user in Task 1. Run interactively:
```
kubectl create secret generic github-oauth \
  --namespace sketchpad \
  --from-literal=client-id='<USER_PROVIDED_CLIENT_ID>' \
  --from-literal=client-secret='<USER_PROVIDED_CLIENT_SECRET>'

FERNET_KEY=$(python3 -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())")
kubectl create secret generic encryption-key \
  --namespace sketchpad \
  --from-literal=fernet-key="$FERNET_KEY"

kubectl create secret generic cloudflared-tunnel-token \
  --namespace sketchpad \
  --from-literal=token='<USER_PROVIDED_TUNNEL_TOKEN>'
```
Verify: `kubectl get secret github-oauth encryption-key cloudflared-tunnel-token -n sketchpad`

**Step 5: Deploy cloudflared**
```
kubectl apply -f k8s/cloudflared/deployment.yaml
```
Verify: `kubectl get pods -n sketchpad -l app=cloudflared` shows Running
Verify: `kubectl logs -n sketchpad -l app=cloudflared --tail=20` shows "connected" messages

**Step 6: Deploy nginx placeholder**
```
kubectl apply -f k8s/placeholder/deployment.yaml
```
Verify: `kubectl get deployment sketchpad-placeholder -n sketchpad` shows 1/1 READY
Verify: `kubectl get svc sketchpad -n sketchpad` shows ClusterIP service

**Step 7: Build and push container image to ghcr.io**
Determine the GitHub username/org for the image name. Build and push:
```
docker build -t ghcr.io/<owner>/sketchpad:latest .
echo $GITHUB_PAT | docker login ghcr.io -u <USERNAME> --password-stdin
docker push ghcr.io/<owner>/sketchpad:latest
```
After push: remind user to set package visibility to Public on GitHub if it defaulted to private.
Note: The placeholder uses nginx directly -- this Dockerfile push is for INFRA-06 verification. Phase 3 will update the Deployment to use this image.

**Step 8: End-to-end verification**
```
curl -sf https://sketchpad.kempenich.ai/
```
Expected: HTTP 200 with JSON body `{"status":"ok","service":"sketchpad","phase":"infrastructure-placeholder"}`

If curl fails with connection error: check cloudflared logs. If 502: verify Service exists and placeholder pod is Running. If DNS error: verify Cloudflare tunnel hostname config.
  </action>
  <verify>
    <automated>kubectl get storageclass nfs-client && kubectl get pvc -n sketchpad -o jsonpath='{range .items[*]}{.metadata.name}={.status.phase}{"\n"}{end}' && kubectl get secret github-oauth encryption-key cloudflared-tunnel-token -n sketchpad && kubectl get deployment sketchpad-placeholder cloudflared -n sketchpad && curl -sf -o /dev/null -w '%{http_code}' https://sketchpad.kempenich.ai/</automated>
  </verify>
  <done>StorageClass nfs-client exists. Both PVCs are Bound. All three Secrets exist. cloudflared and placeholder Deployments are Running. curl to https://sketchpad.kempenich.ai/ returns 200. Container image pushed to ghcr.io.</done>
</task>
