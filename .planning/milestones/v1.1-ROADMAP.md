# Milestone v1.1: Multi-Users

**Status:** SHIPPED 2026-03-07
**Phases:** 5-7
**Total Plans:** 5

## Overview

Each authenticated user gets their own isolated sketchpad, segregated by OAuth username. Includes per-user and global storage limits, plus build tooling migration from Makefile to Justfile with CI gates.

## Phases

### Phase 5: Per-User Storage Isolation

**Goal**: Each authenticated user reads and writes only their own sketchpad, isolated by OAuth username
**Depends on**: Phase 4 (v1.0 shipped)
**Requirements**: ISOL-01, ISOL-02, ISOL-03, ISOL-04
**Plans:** 2 plans

Plans:
- [x] 05-01-PLAN.md -- TDD: user identity resolution module with sanitization and path traversal defense
- [x] 05-02-PLAN.md -- Wire user identity into read/write tools with per-user paths

**Success Criteria:**
  1. Two different GitHub users each see only their own sketchpad content
  2. Path traversal sequences (e.g., `../`) cannot escape the data directory
  3. User directory auto-created on first write
  4. Usernames sanitized to filesystem-safe names
  5. OAuth username never exposed in tool JSON schema

### Phase 6: Storage Limits

**Goal**: Write operations bounded by configurable per-user and global size limits
**Depends on**: Phase 5
**Requirements**: STOR-01, STOR-02
**Plans:** 1 plan

Plans:
- [x] 06-01-PLAN.md -- TDD: per-user and global storage size limits with config migration

**Success Criteria:**
  1. Write exceeding per-user limit (20KB default) rejected with clear error
  2. Write exceeding global limit (50MB default) rejected with clear error
  3. Both limits configurable via environment variables

### Phase 7: Build Tooling Migration

**Goal**: Makefile replaced by Justfile with identical functionality, CI updated
**Depends on**: Phase 4 (independent of Phases 5-6)
**Requirements**: BUILD-01, BUILD-02
**Plans:** 2 plans

Plans:
- [x] 07-01-PLAN.md -- Ruff setup + one-time format + Justfile creation
- [x] 07-02-PLAN.md -- CI workflow update with test/lint gates + Makefile deletion

**Success Criteria:**
  1. Every Makefile recipe has working Justfile equivalent
  2. `just build`, `just deploy`, etc. produce same results as `make`
  3. GitHub Actions CI runs with `setup-just` action
  4. Makefile deleted

## Milestone Summary

**Key Decisions:**
- No slugify library -- GitHub usernames already filesystem-safe; lowercase + regex sufficient
- Assert (not exception) for missing auth -- fail-fast, no fallback to shared storage
- SIZE_LIMIT replaced by MAX_STORAGE_USER (20KB) + MAX_STORAGE_GLOBAL (50MB) -- hard write-time enforcement
- Ruff with E4/E7/E9/F/B/I rules as lint baseline
- Just installed via Homebrew (system tool, not Python dependency)
- Pinned just-version to '1' in CI to prevent major version breakage

**Issues Resolved:**
- None -- clean milestone execution

**Issues Deferred:**
- NFS subdirectory permissions on Synology NAS (needs empirical verification during deployment)

**Technical Debt Incurred:**
- None
