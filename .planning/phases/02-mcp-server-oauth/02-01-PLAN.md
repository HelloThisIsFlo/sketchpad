<task type="auto">
  <name>Task 2: Create FastMCP server with GitHubProvider and file tools</name>
  <files>src/sketchpad/server.py, src/sketchpad/tools.py, src/sketchpad/__main__.py</files>
  <action>
1. Create `src/sketchpad/server.py`:
   - Import config values from sketchpad.config
   - Import register_tools from sketchpad.tools
   - Set up FileTreeStore with FernetEncryptionWrapper for OAuth state persistence:
     ```python
     from key_value.aio.stores.filetree import (
         FileTreeStore,
         FileTreeV1KeySanitizationStrategy,
         FileTreeV1CollectionSanitizationStrategy,
     )
     from key_value.aio.wrappers.encryption import FernetEncryptionWrapper
     from cryptography.fernet import Fernet
     ```
   - Create state_dir Path from STATE_DIR, mkdir(parents=True, exist_ok=True)
   - Instantiate FileTreeStore with state_dir and both sanitization strategies
   - Wrap in FernetEncryptionWrapper using Fernet(STORAGE_ENCRYPTION_KEY)
   - Create GitHubProvider with client_id, client_secret, base_url=SERVER_URL, jwt_signing_key, client_storage=encrypted_store
   - Create FastMCP instance: `mcp = FastMCP(name="Sketchpad", auth=auth)`
   - Call register_tools(mcp) to register tools
   - Add `if __name__ == "__main__"` block: `mcp.run(transport="http", host="0.0.0.0", port=8000)`

2. Create `src/sketchpad/tools.py`:
   - Import os, Path from pathlib
   - Import DATA_DIR, SKETCHPAD_FILENAME, SIZE_LIMIT from sketchpad.config
   - Define SKETCHPAD_PATH = Path(DATA_DIR) / SKETCHPAD_FILENAME
   - Define WELCOME_MESSAGE = "Welcome to Sketchpad! Write something here."
   - Define a `register_tools(mcp)` function that registers both tools on the passed mcp instance:

   `read_file()` tool:
   - Docstring: "Read the sketchpad file. This is a single shared Markdown file for jotting down notes, ideas, and drafts."
   - If file doesn't exist, return WELCOME_MESSAGE
   - Read file content (utf-8)
   - If len(content) > SIZE_LIMIT, append warning: "\n\n---\n[WARNING: File exceeds recommended size limit (~50KB). Consider trimming older content.]"
   - Return content

   `write_file(content: str, mode: str = "replace")` tool:
   - Docstring: "Write to the sketchpad file. Use this for notes, drafts, ideas -- Markdown formatting recommended but not required.\n\nArgs:\n    content: The text to write.\n    mode: 'replace' (default) overwrites the file; 'append' adds to the end."
   - Create parent dirs if needed: SKETCHPAD_PATH.parent.mkdir(parents=True, exist_ok=True)
   - If mode == "append": read existing content (or ""), concatenate, write
   - Else (replace): write content directly
   - Return confirmation: f"File updated ({mode} mode). Size: {SKETCHPAD_PATH.stat().st_size} bytes."

3. Create `src/sketchpad/__main__.py`:
   - This enables `python -m sketchpad.server` and also `python -m sketchpad`:
     ```python
     from sketchpad.server import mcp
     mcp.run(transport="http", host="0.0.0.0", port=8000)
     ```

Important: Use `@mcp.tool` decorator inside the `register_tools` function (tools are closures registered on the passed mcp instance). The RESEARCH.md shows `register_tools(mcp)` as the pattern.
  </action>
